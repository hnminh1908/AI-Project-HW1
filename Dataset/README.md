# Dataset Documentation
## Data type
Time‑series numerical data from the gravity sensor (x, y, z components in $m/s^2$).

Sampling rate $=$ 100 Hz (as recorded by the app). Each raw file contains a continuous recording of about 62 seconds.

## Source
Self‑collected using an iPhone Xs Max and the free application Sensor Logger (available on App Store). No external public dataset was used.

## Conditions for data collection
* **Standing**: Phone held naturally in front of face (slightly tilted), or in trousers pocket, or slightly left/right tilted in hand.
* **Sitting**: Phone placed diagonally relative to ground, or parallel to ground with screen facing up, slightly left/right tilted.
* **Lying down**: User lying on back (phone on chest or beside body, screen horizontal), or lying on side (phone tilted $\approx 90 \degree$).

All recordings were performed in a quiet indoor environment with no rapid movements except standing position (slowly walking)
